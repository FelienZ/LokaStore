# LokaStore

LokaStore adalah aplikasi katalog produk sederhana berbasis web. Proyek ini dibuat untuk latihan mengambil data dari API, menerapkan JavaScript ES6, serta memahami layout menggunakan CSS Grid dan Flexbox.

## Fitur

- Menampilkan kategori produk
- Mengambil dan menampilkan data produk dari FakeStore API
- Layout responsif menggunakan media query
- Tampilan produk dinamis berdasarkan kategori

## Tujuan Pembuatan

Proyek ini dibuat untuk:

- Belajar mengambil dan menampilkan data dari API publik
- Mempraktikkan JavaScript ES6
- Melatih kemampuan layout menggunakan CSS Grid dan Flexbox

## Teknologi

- HTML
- CSS
- JavaScript
- FakeStore API (https://fakestoreapi.com/)

## Rencana Pengembangan

- Modal untuk menampilkan detail produk
- Fitur keranjang belanja
- Backend sederhana
